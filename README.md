Raspberry PI 16x2 Character LCD
=============
![version](https://img.shields.io/badge/version-0.1-orange.svg) 

## How to make and run

    $ sudo cd /pi/home/projects
    $ sudo gcc -o example example.c -lwiringPi -lwiringPiDev X
    $ sudo ./example
